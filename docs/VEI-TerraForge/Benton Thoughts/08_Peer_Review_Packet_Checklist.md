# Peer Review Packet Checklist (IAAO / Academic / Dissertation)

## A. Evidence completeness (required)
- Annual ratio-study metrics table (level + uniformity + vertical equity) with definitions
- Sales validity rule set (decision tree) and exclusion reason taxonomy
- Trimming/outlier policy and sensitivity results
- Stratification policy (class/neighborhood/tier) and rationale
- Holdout protocol (time-slice + random) and performance summary
- Reconciliation rules and audit-trail artifact
- Levy context series and “value ≠ taxes” explanatory insert

## B. Reproducibility (required)
- Data dictionary for every dataset used
- Provenance: source files, extraction scripts, and checksums
- Versioned outputs: tables/figures regenerated from scripts
- Archived intermediate files and logs (no manual-only steps)

## C. Methods expectations (strongly recommended)
- Confidence intervals or bootstrapped uncertainty for key metrics
- Vertical equity: PRD sentinel + tier plots; PRB/slope with uncertainty when microdata exist
- Sales chasing / temporal bias checks (time stratification)
- Robustness checks: by class and by neighborhood/market area

## D. Claims discipline (avoid reviewer rejection)
- If using only published summaries, do not claim causal inference
- If “national rank” is discussed, define the national dataset and harmonization method
- Separate assessment distribution claims from levy policy claims

## E. Manuscript packaging (submission ready)
- 150–250 word abstract
- 5–8 keywords
- 6–10 figures, each with a clear takeaway caption
- Appendix: definitions + microdata field requirements
- One-page practitioner summary (for IAAO)
