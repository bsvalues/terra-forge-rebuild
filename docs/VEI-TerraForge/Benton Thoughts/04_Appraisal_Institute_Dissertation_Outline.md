# 04 — Appraisal Institute Dissertation-Style Manuscript Outline

## Working title
**A Governance Architecture for Mass Appraisal:** Data Hygiene → Sales Validation → Calibration → Reconciliation

## Research problem
Mass appraisal systems fail in predictable ways: biased sales sets, feature drift, overfit models, and weak reconciliation. This dissertation proposes an operational governance architecture that makes these failures detectable and correctable.

## Chapters
1) Literature review: equity, uniformity, vertical equity, governance
2) Data and institutional setting: WA DOR ratio studies; Benton case
3) Measurement system framing: instrument design + error sources
4) The governance stack (modules + audit trails)
5) Empirical evaluation (Benton vs peers; break tests)
6) Implementation specification (software + SOP)
7) Policy implications and replication guidance

## Empirical extensions required
- Sales microdata for PRB and regression slopes
- Parcel feature tables for neighborhood/new-construction decomposition
- Appeals outcomes for “taxpayer friction” validation
