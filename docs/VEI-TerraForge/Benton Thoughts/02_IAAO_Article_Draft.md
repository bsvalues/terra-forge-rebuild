# 02 — IAAO Practitioner Article Draft

## Working title
**Making Vertical Equity Visible:** A Practical Governance Stack for Detecting and Reducing Hidden Regressivity Risk

## Thesis
Counties routinely monitor level and uniformity but under-instrument vertical equity. A lightweight governance stack—data hygiene → sales validation → calibration → reconciliation—makes vertical equity measurable, auditable, and improvable.

## What to publish (short + high impact)
1) A “vertical equity dashboard” pattern (tiers, PRD/PRB, slope tests)
2) A checklist for implementing operational controls
3) A Benton case example showing multi-year stability (WA DOR evidence)

## Practical framework
### 1) Data hygiene module
- Completeness checks, feature drift flags, anomaly triage

### 2) Sales validator module
- Arms-length tests, time adjustments, outlier triage, audit trail

### 3) Calibration module
- Targets, holdouts, stratification, parameter stability

### 4) Reconciliation module
- Human + AI-assisted reconciliation, documented rationale
